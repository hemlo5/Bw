# Boardswallah

Board Exam Papers, Answer Keys & Study Materials

This contains everything you need to run the Boardswallah website locally.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
